# Tarsier v0.1.1 — Design System & Structure Rebuild

> **Goal:** Reshape the existing v0.1 codebase to match the new CLAUDE.md spec. The app currently works but looks default. This phase makes it look and feel like a real language learning app.  
> **Approach:** Update in place — don't nuke working code. Reshape, retheme, restructure.  
> **Reference:** Read CLAUDE.md fully before starting any block. Read docs/Tarsier_Roadmap.md for curriculum context. View the mockup artifact for visual targets.
