# Tarsier v0.3.1 — Onboarding & Lesson Polish

> Continues from v0.3 (Bunso Onboarding + Pronunciation Audio).
> This patch covers UI/UX refinements made during testing and review.

---

## Changes

### Onboarding Architecture

1. **Floating Continue button** — Lifted Continue buttons from 12 individual screen files into `OnboardingFlow.swift` as a persistent bottom element. Button is unaffected by page transition animations and greys out during screen transitions.

2. **Back button + progress bar** — Added chevron-left back button (top-left) and `ProgressBarView` (top-right) to onboarding. Back button hidden on screen 0 (splash). Progress bar hidden on auto-advance and lesson screens.

3. **Direction-aware transitions** — Tracks `isGoingForward` state. Forward navigation slides in from trailing edge; back navigation slides in from leading edge.

4. **Name greeting screen** — After collecting user's name and Ate/Kuya title, added auto-advance screen with Bunso celebrating: "It's nice to get to know you, \<Ate/Kuya\> \<name\>!" Auto-advances after 1.5s or on tap.

5. **No pre-selected answers** — Changed sentinel values: `proficiencyLevel = -1` (was 0), `dailyGoalMinutes = 0` (was 10), `selectedLanguage = ""` (was "Tagalog"). Continue button disabled until user makes a selection.

6. **Superwall temporarily disabled** — `SubscriptionGateScreen` bypasses paywall and calls `onContinue()` directly. SuperwallKit import commented out. Re-enable when dashboard is configured.

### Onboarding Content

7. **Updated proficiency wording** — 5 levels (was different wording):
   - "I'm starting from scratch"
   - "I recognize common words when I hear them"
   - "I understand more than I can say"
   - "I can have basic conversations"
   - "I'm comfortable in most conversations"

8. **Duolingo-style proficiency screen** — Added signal bars icon (ascending bars) to each proficiency option. Bunso mascot with speech bubble asks the question (replaces plain text header). Removed checkmark icons (highlighted fill is sufficient).

9. **Hiligaynon added** — Fourth "Coming soon" language option in `LanguagePickerScreen` (after Cebuano, Ilocano).

10. **Philippine flag PNG** — Replaced 🇵🇭 emoji with `philippine-flag.png` asset (transparent background) across all language picker options.

11. **Vision screen** — Removed card borders from benefit cards for cleaner look.

### Lesson View

12. **Removed teach coach mark** — Deleted "Tap Continue to advance" overlay from `LessonView`. Quiz coach mark still exists.

13. **Updated quiz prompt** — lesson_001.json: Changed "Complete the respectful yes: O___" to "How would you say yes respectfully?"

14. **Vocabulary on session 1** — Words are now added to `WordBankEntry` after completing session 1 (not only after all 5 sessions). This means the Words tab populates immediately after onboarding.

### Home / Roadmap

15. **Flat lesson circles** — Removed dark shadow from `PathNodeView` that made circles look 3D.

16. **"Start!" → "Continue"** — Updated `StartBubbleView` label from "Start!" to "Continue".
