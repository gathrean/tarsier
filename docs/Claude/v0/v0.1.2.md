## 2. Roadmap layout: rows of 1-2 instead of uniform grid

The current home screen shows all lessons in a uniform 2-column grid with names like "Magalang 1, Magalang 2." This needs to change:

- Chapter titles appear as section headers (e.g., "Magalang" with subtitle "Respect & Greetings")
- Lessons appear as circles within rows defined by chapters.json
- Row of 1: single circle, centred
- Row of 2: two circles side by side
- Each circle shows the lesson's own title (e.g., "Po & Opo," not "Magalang 1")
- Circle states:
  - **Locked:** grey background, lock icon
  - **Available:** cream background, lesson number
  - **In progress:** progress ring around circle showing X/5
  - **Completed:** filled functionalPurple with checkmark
- Practice node at the end of each chapter: smaller circle, book icon, labelled "Practice." Unlocks when all lessons in the chapter are complete.
- Lessons unlock sequentially within a chapter. First lesson of each chapter unlocks when previous chapter's Practice is complete. Exception: Chapter 1 Lesson 1 is always unlocked.
- Scroll direction: vertical, same as current

## 3. Lesson names from JSON

Each lesson circle should display the `title` field from the lesson JSON (e.g., "Po & Opo"), not a generated name. Load lesson metadata (id, title, chapter_id) from the lesson JSON files in `Resources/Lessons/`. You don't need to load full session data for the roadmap, just the top-level fields.

## 4. Onboarding change: lesson-first flow

Current flow: welcome → skill picker → motivation → home screen.

New flow: welcome → skill picker → motivation → **straight into Lesson 1 Session 1** → session complete screen (progress ring 1/5, Alam Mo Ba, +15 XP) → **THEN home screen**.

After onboarding, the home screen should show Po & Opo with 1/5 progress already filled. The user has already started their journey before seeing the roadmap.

Implementation:

- Add a flag in SwiftData: `hasCompletedOnboarding: Bool` (default false)
- After motivation picker, if `!hasCompletedOnboarding`: navigate to LessonView for lesson 001, session 1 only (not the full lesson, just session 1)
- After session 1 completes: show the session complete screen, then navigate to home screen and set `hasCompletedOnboarding = true`
- On subsequent app launches: go straight to home screen
