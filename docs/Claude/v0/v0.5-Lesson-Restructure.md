# v0.5 - Lesson Restructure

## What Changed

Moved from flat lesson files to a directory-per-chapter structure with single-session bite-sized lessons.

### Before
```
Resources/Lessons/
  chapters.json
  lesson_000_intro.json
  lesson_001.json
  lesson_002.json
  ...
  lesson_032.json
```
- 34 flat JSON files in one directory
- Multi-session schema (total_sessions: 5, multiple sessions per lesson)
- Old chapter names (Respect & Greetings, Food & Basic Verbs, etc.)

### After
```
Resources/Lessons/
  chapters.json
  ch00_introduction/
    lesson_000.json
  ch01_hello/
    lesson_001.json  (How Are You?)
    lesson_002.json
    ...
    lesson_012.json  (Chapter Review)
```
- Directory per chapter, lessons grouped inside
- Single-session schema (total_sessions: 1, one session per lesson)
- New `lesson_type` field: "teaching", "review", "chapter_review"
- Shorter chapter names: Hello, Food, Family, Feelings, Travel, Plans, Affixes

## Files Modified

### Data
- `Resources/Lessons/chapters.json` - Added `directory` field, `total_lessons`, `total_words`. Ch01 has 12 lessons. Ch02-10 have empty rows (stubs).
- `Resources/Lessons/ch00_introduction/lesson_000.json` - Moved from root
- `Resources/Lessons/ch01_hello/lesson_001-012.json` - 12 new single-session lessons

### Models
- `Models/SlideLesson.swift` - Added `positionInChapter: Int?` and `lessonType: String?`
- `Models/Chapter.swift` - Added `directory: String?`, `totalLessons: Int?`, `totalWords: Int?`

### Services
- `Services/LessonService.swift` - Now scans chapter subdirectories instead of flat bundle root. Reads `chapters.json` for directory names, then loads `lesson_*.json` from each `ch{NN}_*` subdirectory. Sorts by chapter ID then `position_in_chapter`.

## What's NOT Touched
- SwiftData models (UserProfile, LessonProgress, SessionProgress, etc.)
- Gemini AI service
- RevenueCat / Superwall
- Navigation / tab structure
- Vocab tab / Me tab
- Onboarding flow
- All SwiftUI views (they consume the same SlideLesson/Chapter types)

## New JSON Schema Fields

```json
{
  "lesson_type": "teaching",       // "teaching" | "review" | "chapter_review"
  "position_in_chapter": 1,        // ordering within chapter
  "total_sessions": 1              // always 1 in new format
}
```

chapters.json additions:
```json
{
  "directory": "ch01_hello",       // subdirectory name
  "total_lessons": 12,             // lesson count for display
  "total_words": 25                // vocab count for display
}
```
